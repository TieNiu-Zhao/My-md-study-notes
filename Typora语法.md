# 常用语法

````c++
# + 空格				//标题（几个#就是几级标题）
- + 空格				//列表（在列表状态下按两下回车换行）
* + 文字 + *			//斜体
** + 文字 + **		//粗体
~~ + 文字 + ~~ 		//删除线
---					//分割线
```					//代码块
>					//引用（可嵌套引用）
[名字] + (网址)		  //超链接
````
